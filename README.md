# Gestor de Tareas

Este proyecto es una aplicación de terminal para gestionar tareas.

## Funcionalidades
- Listar tareas.
- Añadir nuevas tareas.
- Eliminar tareas existentes.

## Requisitos
- Python 3.8 o superior.
